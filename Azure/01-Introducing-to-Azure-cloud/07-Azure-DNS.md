4. What is a primary benefit of Azure DNS?
You can manage DNS records using Azure credentials and tools.


**Azure DNS** is a hosting service for DNS domains that provides name resolution by using Microsoft Azure infrastructure. 

`Azure DNS` uses the scope and scale of `Microsoft Azure` to provide numerous benefits, including:
- Reliability and performance
- Security
- Ease of use
- Customizable virtual networks
- Alias records

DNS domains in `Azure DNS` are hosted on Azure's global network of DNS name servers, providing resiliency and high availability. `Azure DNS` uses a`nycast networking`, so the closest available DNS server answers each DNS query, providing fast performance and high availability for your domain.

Azure DNS is based on Azure Resource Manager, which provides features such as Azure role-based access control (Azure RBAC), activity logs, and resource locks to help protect DNS resources and records.
Azure DNS is integrated in the Azure portal and uses the same credentials, support contract, and billing as your other Azure services.

You can't use Azure DNS to buy a domain name. For an annual fee, you can buy a domain name by using App Service domains or a third-party domain name registrar. Once purchased, your domains can be hosted in Azure DNS for record management.
